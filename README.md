# Amazon Affiliate Landing Page - সম্পূর্ণ গাইড

এটি একটি সহজ, আধুনিক এবং এক-পৃষ্ঠার Amazon অ্যাফিলিয়েট ওয়েবসাইট যা **HTML, CSS এবং JavaScript** দিয়ে তৈরি।

## 🚀 বৈশিষ্ট্য

✅ **প্রোডাক্ট ম্যানেজমেন্ট** - সহজেই Amazon প্রোডাক্ট যোগ করুন\
✅ **ক্লিক ট্র্যাকিং** - কতজন কত ক্লিক করেছে তা দেখুন\
✅ **ক্যাটাগরি ফিল্টার** - বিভিন্ন ক্যাটাগরিতে প্রোডাক্ট সাজান\
✅ **অ্যানালিটিক্স** - টপ প্রোডাক্ট, ক্লিক ট্রেন্ড দেখুন\
✅ **Facebook শেয়ারিং** - Facebook-এ সরাসরি শেয়ার করুন\
✅ **মোবাইল রেসপন্সিভ** - সব ডিভাইসে নিখুঁত দেখায়\
✅ **দ্রুত লোড** - কোন বাহ্যিক লাইব্রেরি নেই, শুধু vanilla JS\
✅ **LocalStorage ডেটা** - সব ডেটা স্থানীয়ভাবে সংরক্ষিত থাকে

## 📁 ফাইল কাঠামো

```
Amzon-Affiliate/

└── README.md       # এই ফাইল
```

## 🔧 কীভাবে ব্যবহার করবেন

### 1. **ওয়েবসাইট খুলুন**
`index.html` কে আপনার ব্রাউজারে খুলুন:
```
Double-click on index.html
```

### 2. **প্রোডাক্ট যোগ করুন**
- **প্রোডাক্ট নাম** - যেমন: "Wireless Headphones"
- **দাম** - স্থানীয় মুদ্রায় (৳)
- **Amazon লিংক** - আপনার অ্যাফিলিয়েট লিংক যা amazon.com থেকে শুরু হয়
- **ক্যাটাগরি** - Electronics, Fashion, Home, Books, Sports ইত্যাদি
- **ইমেজ URL** - প্রোডাক্টের ছবির লিংক (ঐচ্ছিক)
- **বিবরণ** - ছোট ডেসক্রিপশন

তারপর "Add Product" ক্লিক করুন।

### 3. **প্রোডাক্ট দেখুন**
সব প্রোডাক্ট সুন্দর কার্ড-এ প্রদর্শিত হবে।

### 4. **ফিল্টার করুন**
ক্যাটাগরি বাটন ক্লিক করে প্রোডাক্ট ফিল্টার করুন।

### 5. **ক্লিক ট্র্যাকিং**
- গ্রাহক যখন "Buy Now" ক্লিক করবে, তা স্বয়ংক্রিয়ভাবে ট্র্যাক হবে
- ক্লিক সংখ্যা, কেটাগরি ক্লিক, টপ প্রোডাক্ট সব দেখা যাবে

### 6. **অ্যানালিটিক্স**
পৃষ্ঠার নিচে অ্যানালিটিক্স ড্যাশবোর্ড দেখুন:
- **Total Clicks** - মোট ক্লিক
- **Today's Clicks** - আজকের ক্লিক
- **Top Product** - সবচেয়ে বেশি ক্লিক হওয়া প্রোডাক্ট
- **Most Active Category** - কোন ক্যাটাগরিতে সবচেয়ে বেশি ক্লিক

### 7. **Facebook-এ শেয়ার করুন**
"Share on Facebook" বাটন ক্লিক করুন এবং আপনার বন্ধুদের সাথে শেয়ার করুন।

## 🌐 ডেটা সংরক্ষণ

সব ডেটা আপনার ব্রাউজারের **LocalStorage**-এ সংরক্ষিত থাকে। এটি মানে:
- ব্রাউজার বন্ধ করলেও ডেটা থাকবে
- সব কম্পিউটারে ডেটা আলাদা থাকবে
- ব্রাউজার ডেটা ডিলিট করলে সব হারিয়ে যাবে

## 🎨 কাস্টমাইজেশন

### রঙ পরিবর্তন করতে:
`style.css` খুলুন এবং এই রঙগুলি পরিবর্তন করুন:
```css
--primary-color: #FF9500;        /* মূল রঙ (কমলা) */
--secondary-color: #146EB4;      /* দ্বিতীয় রঙ (নীল) */
--text-color: #333;              /* টেক্সট রঙ */
```

### লোগো পরিবর্তন করতে:
`index.html` খুলুন এবং এই লাইনে লোগো বদলান:
```html
<h1 class="logo">🛍️ Best Deals</h1>
```

### Facebook মেটা ট্যাগ আপডেট করতে:
`index.html` এর head-এ এই লাইনগুলি সম্পাদনা করুন:
```html
<meta property="og:title" content="আপনার সাইটের নাম">
<meta property="og:description" content="আপনার বর্ণনা">
<meta property="og:image" content="আপনার ছবির URL">
```

2. F12 বা Cmd+Option+I (Mac) দিয়ে DevTools খুলুন
3. Console-এ লিখুন: `copy(JSON.stringify(localStorage))`
4. ডেটা কপি হয়ে যাবে
